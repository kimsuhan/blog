# Task 27. MVP 통합 점검

## 목표

MVP 완료 기준을 하나씩 검증하고 빠진 부분을 정리한다.

## 작업 내용

- 개인 도메인 접속 가능 여부 확인
- Markdown 게시글 작성 가능 여부 확인
- 관리자 API 게시글 생성 가능 여부 확인
- draft/published/archived 상태 확인
- PostgreSQL metadata 저장 확인
- 태그 페이지 확인
- 내부 링크와 백링크 확인
- sitemap, RSS, 기본 SEO 확인
- Docker Compose 재배포 확인

## 수정 예상 파일

- `docs/mvp-roadmap.md`
- 발견된 결함에 해당하는 코드 파일

## 하지 말 것

- Phase 2 기능을 섞어서 진행하지 않기
- MVP 제외 기능을 끌어오지 않기

## 완료 기준

- `docs/mvp-roadmap.md`의 MVP 완료 기준을 모두 체크 가능
- 남은 작업은 Phase 2 또는 버그로 분류
