# Task 28. Phase 2 계획 재정리

## 목표

MVP 완료 후 진행할 Phase 2 작업을 실제 운영 경험에 맞춰 재정리한다.

## 작업 내용

- draft preview 페이지 설계
- 검색 인덱스 고도화 방향 결정
- Pagefind 또는 Meilisearch 선택 기준 정리
- API 요청 로그 구현 범위 결정
- 게시글 수정 이력 범위 결정
- 자동 백업 스크립트 구현 여부 결정

## 수정 예상 파일

- `docs/mvp-roadmap.md`
- 필요 시 `docs/phase2.md`

## 하지 말 것

- MVP 중간에 Phase 2 구현 시작하지 않기

## 완료 기준

- Phase 2 우선순위가 문서화됨
- MVP와 Phase 2 경계가 명확함

## 구현 메모

- `docs/phase2.md`를 추가해 Phase 2 우선순위와 각 작업의 범위를 정리했다.
- `docs/mvp-roadmap.md`의 Phase 2 섹션은 상세 문서 링크와 요약 표로 정리했다.
- 자동 백업 스크립트, Nginx, DB, Cloudflare 같은 운영 작업은 사용자가 직접 관리하는 항목으로 분리했다.
